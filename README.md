# Lord of the Rings Filming Locations — Wellington, NZ

An interactive map of 13 Lord of the Rings filming locations in and around Wellington, New Zealand, with photos, scene descriptions, accessibility info, and a suggested day-trip driving route.

**Live site:** [jeffreyruoss.github.io/wellington-nz](https://jeffreyruoss.github.io/wellington-nz/)

> **Disclaimer:** This site was generated with AI assistance. Location details, directions, and access information may contain errors or become outdated. Please verify details before visiting.
